Interactive / Preemptive Scheduling


In pre-emptive scheduling a process will execute until it is interrupted

Priority Scheduling


A Priority based CPU- Scheduling Scheme may be implemented by having a separate queue for each priority level reported by the Operating System. When the priority queues are parsed highest priority queues have processes taken from them first.

(Sibsankar Haldar, Alex A. Aravind 2009, 123)

For each priority queue all processes in that level are accessed via a FCFS scheduling mechanism. In simple terms the priority implementation is a concatenation of FCFS queues for all the different priority levels.

(Sibsankar Haldar, Alex A. Aravind 2009, 123)
